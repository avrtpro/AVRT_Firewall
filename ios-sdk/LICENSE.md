## Data and Privacy

### Data Processing
- AVRT processes text and voice transcriptions for validation purposes
- All processing adheres to privacy-by-design principles
- No personal data is retained beyond validation session unless audit logging is enabled

### Audit Logging
- When enabled, audit logs are encrypted using AES-256
- Logs contain: request ID, timestamps, validation scores, and hashed content
